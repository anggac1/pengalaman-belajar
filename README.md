## Petunjuk untuk running model <br>
1. Upload terlebih dahulu  kaggle.json yang dikirim sebagai API (atau bisa menggunakan API sendiri)
2. Tidak perlu untuk upload data, data diimpor di dalam kode ipynb
3. Waktu run bisa 30 menit atau lebih, jika ingin mengoreksi hasil output seharusnya sudah tersedia di notebook
